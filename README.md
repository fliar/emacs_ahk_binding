# Emacs ahk binding
This script simply binds emacs hotkey to target
## Usage
tested with VS2016
Add your emacs target to IsTarget
*Shift+CapsLock* to enable emacs mode
## Current Bindings
c-x c-f 		: ctrl-o
c-x k 			: ctrl-f4
c-x u 			: ctrl-z
c-f 			: right
c-b 			: left
c-n 			: down
c-p 			: up
c-g 			: Esc
m-@ 			: Mark toggle
c-a 			: Home Home
m-m 			: End Home
c-e 			: End
c-w 			: ctrl-x
m-w 			: ctrl-c
c-y 			: ctrl-v
c-k 			: shift-End ctrl-x
m-g m-g 		: ctrl-g
